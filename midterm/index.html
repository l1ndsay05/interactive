Among Us Parody Game

Midterm Project Overview

Welcome to our midterm project page! Over the past month, we have designed a fully functioning single-player Among Us parody game, incorporating unique interpretations, mini-games, and imposters.

Project Features

We aimed to create a user-controlled character complete with walking animations, achieved through a sprite sheet divided into cells, each representing a frame. By iterating through these frames, we implemented various walking animations based on the direction the user controls.

Our character navigates an Among Us-inspired map. When a directional key is pressed, the map scrolls accordingly, keeping the character centered. A Hitmap beneath the map limits character and NPC movement via color detection.

Mini-Games

Each team member contributed a unique Among Us-themed mini-game, integrating p5.js features like collision detection, parallax scrolling, particle systems, and Perlin noise. Each mini-game activates when the player is within proximity, enhancing the interactive experience.

Game Functions

To play the game, user can click on on WASD to control the user character. Player must avoid all NPC's in order to stay alive, and play all 3 mini task mini games to win this game.
